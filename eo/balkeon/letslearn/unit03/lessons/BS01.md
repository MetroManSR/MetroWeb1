# Orden de la Oración

En Balkeon, la oración se ordena de la siguiente manera:

### Declarativas (Afirmativas)

**S-V-O**  
**Sujeto-Verbo-Objeto**

Este orden se utiliza para todos los casos afirmativos.

### Interrogativas

**V-S-O**  
**Verbo-Sujeto-Objeto**

Este orden se utiliza para formar preguntas.

---

## Ejemplos

### Afirmativas
- **Yo no iré** = Aye gonib nek
- **Ella comió** = A namnip

### Interrogativas
- **¿Irás tú?** = Gonib yu?
- **¿Comió ella?** = Nanmip a?
